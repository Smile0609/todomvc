  + ng-class
- 添加任务
- 删除任务
- 显示未完成任务数量
- 点击清除所有已完成任务

- 点击显示已完成任务

## ng调试

SimpleExtManager Chrome 用来管理插件的插件

- 给 window 暴露 $scope
- 使用 Chrome 插件 Angular Batarang
  + 通过 Chrome 应用程序商店搜索该插件名称，然后安装
  + 以http协议访问你的要调试的项目
  + 调试之前要记得启用该插件
  + 打开控制台
